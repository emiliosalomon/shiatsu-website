Gutscheine

Machen Sie Ihren Lieben eine Freude mit einem Shiatsu‑ oder Cranio Sacrale‑Gutschein. So funktioniert es:

  1. Wählen Sie den gewünschten Betrag oder eine Einheit (siehe Preise).
  2. Klicken Sie auf den Button unten, um einen Gutschein als PDF auszudrucken (Browser-Druckdialog).
  3. Kontaktieren Sie mich telefonisch oder per E‑Mail, um den Gutschein persönlich abzustimmen.
  4. Überweisen Sie den Betrag an das unten angegebene Konto.
